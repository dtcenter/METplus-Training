====================
Training Environment
====================

Each of the training modules consists of a video recording of the content followed by the script for that video, including all commands executed on the command line. Users are encouraged to setup their own training environment and follow along with these exercises. Commands listed in code blocks, as shown below, may be copied and pasted directly into your training environment!

.. code-block:: none

  echo "Welcome to METplus!"
